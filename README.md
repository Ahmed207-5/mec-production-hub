# إنتاج Hub — MDP Archive

أرشيف مركزي لروابط Google Drive الخاصة بدفعات قسم التصميم والإنتاج الميكانيكي – هندسة الزقازيق، منظم حسب الدفعة والسنة الدراسية.

الموقع لا يستضيف أي ملفات — فقط ينظم الروابط.

## التشغيل محليًا

```bash
npm install
npm run dev
```

## إضافة/تعديل روابط الدرايف

عدّل الملف: `src/data/batches.ts`

## إعداد نموذج "الإبلاغ عن رابط" / "إضافة رابط"

عدّل `GOOGLE_FORM_URL` في: `src/config/site.ts`

## النشر

جاهز للنشر مباشرة على Vercel (لا حاجة لأي متغيرات بيئة أو قاعدة بيانات).
